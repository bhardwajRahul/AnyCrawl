# การค้นหาเว็บและการดึงข้อมูลด้วย MCP ใน Windsurf

เพิ่มความสามารถด้านการค้นหาและการดึงข้อมูลเว็บใน Windsurf ด้วย AnyCrawl MCP

## การตั้งค่าแบบเร็ว

### 1. รับ API Key

ลงทะเบียนที่ [anycrawl.dev](https://anycrawl.dev) แล้วคัดลอก API key จากแดชบอร์ด

### 2. เพิ่มใน Windsurf

เพิ่มสิ่งนี้ใน `~/.codeium/windsurf/mcp_config.json`:

```json
{
    "mcpServers": {
        "anycrawl": {
            "command": "npx",
            "args": ["-y", "anycrawl-mcp"],
            "env": {
                "ANYCRAWL_API_KEY": "your-api-key"
            }
        }
    }
}
```

แทนที่ `your-api-key` ด้วย AnyCrawl API key จริงของคุณ

### 3. รีสตาร์ท Windsurf

เสร็จแล้ว Windsurf สามารถค้นหาและดึงข้อมูลเว็บได้

## สาธิตแบบเร็ว

ลองคำสั่งเหล่านี้ใน Windsurf:

**ค้นหาเว็บ:**

```
Search for the latest Tailwind CSS features
```

**ดึงข้อมูลหน้าเว็บ:**

```
Scrape anycrawl.dev and explain what it does
```

**ดึงเอกสาร:**

```
Find and scrape the Supabase authentication documentation
```

**ครอลเว็บไซต์:**

```
Crawl the docs section of nextjs.org and summarize the key concepts
```

เอเจนต์ AI ของ Windsurf จะใช้เครื่องมือ AnyCrawl โดยอัตโนมัติ

## แก้ปัญหา

### เซิร์ฟเวอร์ไม่ตอบสนอง

1. ตรวจสอบว่า API key ใช้ได้
2. ติดตั้ง Node.js 18 ขึ้นไป
3. รีสตาร์ท Windsurf หลังแก้การตั้งค่า

### ข้อผิดพลาดสิทธิ์

ให้แน่ใจว่า npx รันได้ อาจต้องติดตั้งแพ็กเกจแบบ global:

```bash
npm install -g anycrawl-mcp
```

จากนั้นอัปเดตการตั้งค่าให้ใช้แพ็กเกจ global:

```json
{
    "mcpServers": {
        "anycrawl": {
            "command": "anycrawl-mcp",
            "env": {
                "ANYCRAWL_API_KEY": "your-api-key"
            }
        }
    }
}
```
