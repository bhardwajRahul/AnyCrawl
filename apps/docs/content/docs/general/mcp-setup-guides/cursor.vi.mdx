# Tìm kiếm web và Scrape MCP trong Cursor

Thêm khả năng thu thập web và tìm kiếm vào Cursor với AnyCrawl MCP.

## Thiết lập nhanh

### 1. Lấy API key

Đăng ký tại [anycrawl.dev](https://anycrawl.dev) và sao chép API key từ bảng điều khiển.

### 2. Thêm vào Cursor

Mở Settings (`Cmd+,` trên Mac, `Ctrl+,` trên Windows), tìm "MCP", và thêm:

```json
{
    "mcpServers": {
        "anycrawl": {
            "command": "npx",
            "args": ["-y", "anycrawl-mcp"],
            "env": {
                "ANYCRAWL_API_KEY": "your-api-key"
            }
        }
    }
}
```

Thay `your-api-key` bằng API key AnyCrawl thực tế của bạn.

### 3. Khởi động lại Cursor

Xong! Bạn có thể tìm kiếm và thu thập web ngay trong Cursor.

## Demo nhanh

Thử trong Cursor Chat (`Cmd+K`):

**Tìm kiếm web:**

```
Search for TypeScript best practices 2025
```

**Scrape một trang:**

```
Scrape anycrawl.dev and tell me what it does
```

**Lấy tài liệu:**

```
Scrape the React hooks documentation and explain useEffect
```

**Crawl một website:**

```
Crawl the blog section of example.com and summarize the articles
```

Cursor sẽ tự động dùng các công cụ tìm kiếm, scrape và crawl của AnyCrawl.

## Xử lý sự cố

### Server không phản hồi

1. Kiểm tra API key có hợp lệ không
2. Đảm bảo đã cài Node.js 18+
3. Khởi động lại Cursor sau khi đổi cấu hình

### Lỗi quyền

Đảm bảo npx có quyền thực thi. Có thể cần cài gói toàn cục:

```bash
npm install -g anycrawl-mcp
```

Sau đó cập nhật cấu hình để dùng gói toàn cục:

```json
{
    "mcpServers": {
        "anycrawl": {
            "command": "anycrawl-mcp",
            "env": {
                "ANYCRAWL_API_KEY": "your-api-key"
            }
        }
    }
}
```
